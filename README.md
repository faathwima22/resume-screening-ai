# Resume Screening AI

## Description
This project is an AI-based system that screens resumes and predicts whether a candidate is **Selected** or **Not Selected** using Machine Learning and text classification techniques.

## Features
- Resume text classification using ML
- TF-IDF vectorization for text processing
- Naive Bayes classifier for prediction
- Continuous user input for multiple resume screening
- Beginner-friendly and easy to understand

## Dataset
A sample dataset is created within the code containing:
- Resume text samples
- Labels: selected / not selected

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF Vectorizer)

## Steps
1. Import libraries  
2. Create/load dataset  
3. Encode labels (selected = 1, not selected = 0)  
4. Split dataset into training and testing  
5. Convert text into numerical vectors using TF-IDF  
6. Train Naive Bayes classifier  
7. Evaluate model performance  
8. Take user input and predict selection
