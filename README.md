# resume-screening-ai
Resume Screening AI Developed a machine learning-based resume classification system that analyzes and categorizes resumes into job domains using Natural Language Processing (NLP) techniques. Implemented text preprocessing and TF-IDF vectorization, and trained a Logistic Regression model to improve classification accuracy.
# Resume Screening AI

## 📌 Overview
This project is an AI-based Resume Screening System that classifies resumes into job categories using Machine Learning and NLP.

## 🚀 Features
- Resume classification
- Uses Hugging Face dataset
- Streamlit web app
- Easy to use interface

## 🧠 Tech Stack
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit
- Hugging Face Datasets

## 📊 How it Works
1. Load dataset from Hugging Face
2. Preprocess resume text
3. Convert text to vectors using TF-IDF
4. Train Logistic Regression model
5. Predict category of new resume
